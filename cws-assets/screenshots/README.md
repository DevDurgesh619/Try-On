Easiest capture flow:

1. Install the unpacked extension from `extension/dist/` in Chrome.
2. Resize the browser to fit the 1280×800 frame (or take a wide screenshot
   and crop in Preview / Photos).
3. macOS: `Cmd-Shift-4` then space to capture a window. Linux/Windows: any
   screenshot tool with a region selector.
4. Save here.

Don't include personally identifying details in the demo screenshots —
swap your real face for a stock photo or AI-generated portrait if you'd
rather not have your reference photo on the Web Store.
